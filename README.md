## Description

[Nest](https://github.com/nestjs/nest).js example Web Service. 

- [X] Swagger Documentation
- [X] JWT Authentication (Acess-Refresh token)
- [X] Role Based Authorization
- [X] E2E (Integration) tests

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
